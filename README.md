# WeatherSage 🌤️

**WeatherSage** is a fully responsive **React.js weather application** built with the **OpenWeatherMap API**. It provides **accurate local and global weather forecasts**, supports **geolocation detection** with a fallback to a default city, and features a polished **light** and **dark mode UI**. Fast, scalable, and perfect for modern weather tracking.

---

## Features

- **Accurate Weather Forecasts**: Get current weather, daily and weekly forecasts.
- **Geolocation Detection**: Automatically detects your location for instant weather info.
- **Fallback to Default City**: Ensures app always works even if location permission is denied.
- **Dark Mode UI**: Sleek light/dark theme toggle for comfortable viewing.
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices.
- **Scalable Architecture**: Built with clean code and optimized for future features.
